# TDARAB Frontend

[![React](https://img.shields.io/badge/React-20232A?logo=react&logoColor=61DAFB)]()
[![Vite](https://img.shields.io/badge/Vite-646CFF?logo=vite&logoColor=white)]()

Student-facing web application for the TDARAB medical education platform. Built with React and Vite.

## Overview

This is the frontend where students browse the curriculum, answer MCQ questions, track their progress, and activate their subscriptions.

## Tech Stack

- React
- Vite
- React Router
- Axios
